Para compilar y ejecutar el programa realice los siguientes pasos:

1. Abrir la carpeta desde el cmd (Símbolo del sistema).
2. Escriba "cd src" para cambiar de carpeta y presione "Enter".
3.Escriba "cd test" para cambiar de carpeta y presione "Enter".
4. Para compilar escriba el comando "javac App.java" y presionar "Enter".
5. Despúes, para ejecutar escriba el comando "java App" y presionar "Enter".
